# Register And Login Form (MERN)🖋️🖊️👩‍💻 :-
## Installation : 
### cd client <br/>
> npm install react-router-dom bootstrap react-dom axios vite <br/>
> npm run dev (to check output )<br/>
### cd server <br/>
> npm install nodemon <br/>
> npm start (to check the server listening )<br/>
